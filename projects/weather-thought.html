The Weather Thought

React, Firebase, OpenWeather API

the unit vh stands for the unit view height which is the total width of the actual window view height. This works great on a desktop, but for mobile, there are additional bars that usual collapse when you start scrolling. That small line isn't taken into account when calculating the view height. You can account for that small amount by using the unit svh. This stands for small view height, which does take into account that it might be viewed on mobile. This adds to mobile experience.

About this project

This blog style website is run with React. It utilizes a Firebase database and authentication.

It started out purely as a blog, then I thought, added a widget would be fun, so I used the OpenWeather API to allow users to save their location so that it will display their hometowns current weather.
